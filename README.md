# ⏰ 운동알람 (cutework_alram)

지정된 시간에 알림을 받고, 미리 등록해둔 **유튜브 운동 영상을 원터치로 실행**하는 홈트레이닝 알람 PWA입니다.

> 운동 의지는 있는데 매번 영상을 다시 검색하느라 시작이 늦어지나요?
> 운동알람이 정해진 시간에 알려주고, 알림을 누르면 유튜브 앱에서 영상이 바로 재생됩니다.

## ✨ 주요 기능

- **매주 반복 알람** — 요일(복수)·시간을 정하면 매주 자동 반복
- **3단계 간편 등록** — 시간·요일 → 유튜브 링크 → 알람명
- **원터치 재생** — 알림 클릭 시 유튜브 앱 딥링크 실행, 미설치 시 브라우저로 자동 폴백
- **영상 미리보기** — 링크 등록 시 썸네일·제목·채널명 자동 조회 (실패 시 대체 일러스트)
- **재알림(스누즈)** — 알림을 놓치면 5분 간격 최대 3회 재알림, [종료] 버튼으로 완전 중단 가능
- **NEXT WORKOUT 배너** — 48시간 이내면 실시간 카운트다운, 이후 일정은 알람명·요일·시간 표기
- **요일별 보기** — 상단 날짜를 눌러 해당 요일의 알람만 확인 (시작 시 오늘 자동 선택)
- **알람 관리** — 수정, 삭제(확인 모달), 켜기/끄기, 8가지 파스텔 고유 색상
- **PWA** — 홈 화면 설치, 오프라인 실행, 로그인·서버 없이 모든 데이터 로컬 저장

## 🚀 설치 방법

서비스 워커와 알림 기능은 **HTTPS 환경**이 필요합니다. GitHub Pages로 무료 배포하는 방법을 권장합니다.

### 1) GitHub Pages로 배포

1. 이 저장소를 fork 하거나, 새 저장소를 만들어 위 파일들을 저장소 **루트**에 업로드합니다.
2. 저장소의 **Settings → Pages** 로 이동합니다.
3. **Source** 를 `Deploy from a branch`, Branch 를 `main` / `(root)` 로 설정하고 저장합니다.
4. 잠시 후 발급되는 주소로 접속합니다.
   `https://<username>.github.io/<repository>/`

### 2) Android 홈 화면에 설치

1. Android **Chrome** 에서 배포된 주소에 접속합니다.
2. 우측 상단 메뉴(⋮) → **"홈 화면에 추가"** (또는 "앱 설치") 를 선택합니다.
3. 홈 화면의 ⏰ 아이콘으로 실행하면 주소창 없는 독립 앱처럼 동작합니다.

### 3) 로컬에서 실행해보기 (개발용)

```bash
# 저장소 클론
git clone https://github.com/<username>/<repository>.git
cd <repository>

# 간단한 로컬 서버 실행 (둘 중 하나)
python3 -m http.server 8000
# 또는
npx serve .
```

브라우저에서 `http://localhost:8000` 접속. (localhost는 HTTPS 없이도 서비스 워커가 동작합니다.)

## 📖 사용법

### 알람 등록
1. 하단 **SET WORK OUT 💪** 버튼을 누릅니다.
2. **STEP 1** — 운동 시간과 요일을 선택합니다. 선택한 요일마다 매주 반복돼요.
3. **STEP 2** — 유튜브 영상 링크를 붙여넣고 [등록]을 누르면 미리보기가 표시됩니다.
   (`watch`, `youtu.be`, `shorts` 링크 지원)
4. **STEP 3** — 알람명을 입력하고 [운동알람 등록]을 누르면 완료!

### 알림이 울리면
| 버튼 | 동작 |
|---|---|
| ▶ 영상 재생하기 | 유튜브 실행 + 재알림 중단 |
| 5분 뒤 다시 알림 | 5분 후 재알림 (최대 3회) |
| 종료 | 오늘 이 알람의 재알림 완전 중단 |

### 알람 관리
- 상단 **날짜**를 누르면 해당 요일의 알람만 볼 수 있어요. 같은 날짜를 다시 누르면 전체 보기.
- 각 알람 카드에서 **[삭제 | 수정 | ▶ 재생]** 버튼과 켜기/끄기 스위치를 사용할 수 있어요.
- 첫 실행 시 상단의 **알림 권한** 배지를 눌러 권한을 허용해주세요.

## ⚠️ 알아두기

- **알림 발송 조건** : 웹 플랫폼 특성상 앱(탭)이 실행 중이거나 백그라운드에 살아있을 때 알림이
  발송됩니다. 완전히 종료된 상태의 알람이 필요하면 앱을 홈 화면에 설치하고 사용 시간대에 실행해두세요.
- **데이터 저장** : 모든 데이터는 기기 브라우저(localStorage)에만 저장됩니다. 브라우저 데이터를
  삭제하면 알람도 함께 삭제됩니다. 서버 전송·수집은 없습니다.
- **유튜브 정책** : 본 앱은 유튜브 공식 앱 또는 웹을 통해서만 영상을 재생하며, 콘텐츠를
  저장·변형하거나 광고를 제거하지 않습니다.

## 🛠 기술 스택

- Vanilla HTML / CSS / JavaScript (프레임워크·빌드 없음)
- Web App Manifest + Service Worker (PWA)
- Notification API, Vibration API
- YouTube oEmbed (영상 메타 조회, noembed 폴백)
- Android Intent 딥링크 (유튜브 앱 우선 실행)
